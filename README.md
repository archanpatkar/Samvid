# संविद् 
#### `संविद् (Samvid) means Scheme in Sanskrit`
###  A Scheme Compiler

### Examples


###### Scheme
```scheme
(begin (define r 10) (* pi (* r r)))
```

###### Output
```scheme
314.1592653589793
```

###### Scheme
```scheme
(define hello (lambda (msg) (print (+ (quote Hello,) msg))))
(hello (quote Archan))
```

###### Output
```scheme
Hello,Archan
```
